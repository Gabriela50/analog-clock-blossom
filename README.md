# Analog Clock Blossom

## Descripción

Analog Clock Blossom es una aplicación de escritorio desarrollada en Python que implementa un reloj analógico con tres manecillas: hora, minuto y segundo.

El proyecto aplica programación orientada a objetos, interfaz gráfica con PySide6, base de datos SQLite para guardar configuraciones del usuario y una lista circular doble para representar las marcas del reloj.

La interfaz fue diseñada con un estilo visual pastel y girly, manteniendo una estructura profesional y organizada.

## Requisitos del taller cumplidos

- Aplicación desarrollada completamente en Python.
- Interfaz gráfica o frontend usando PySide6.
- Reloj analógico con tres manecillas.
- Programación orientada a objetos.
- Código, carpetas, archivos, clases, métodos y variables en inglés.
- Textos visibles para el usuario en español.
- Base de datos SQLite para guardar personalización.
- Uso de listas circulares dobles.
- Ejecución local en el portátil.
- Funcionalidad adicional: recordatorio configurable.
- Interfaz visual moderna, personalizada y organizada.
